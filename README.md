## Overview

The Official SelfKey Identity Wallet for Desktop

## Prerequisites

* [NodeJS](https://nodejs.org) version 9.5 or better.

### Install Dependencies

    npm install

### Build on OSX/macOS/Linux

    npm run make

### Build on Windows

    npm run makewin

## Development

### Run the App on OSX/macOS/Linux

    npm run start

### Run the App on Windows

    npm run startwin

## Tests

### Linting

    npm run lint

### Tests

    npm run test
